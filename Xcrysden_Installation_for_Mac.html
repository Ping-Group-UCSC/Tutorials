Xcrysden installation for Mac(M2)

@Shimin Zhang

Install from port

System: MacOS M2 chip;

Instruction from official website

  1. install port;

    if you’re the newest version ; • macOS Sonoma v14

    source ~/.bash_profile after installation.

  1. download and install Quartz. This is the graphic interface support.
  1. try to use port to install xcryden
    sudo port install xcrysden

    port will download all the dependent library.

    after installation,

  1. try to run xcrysden
    xrysden

This step works for me. If there’re error message saids ‘didn’t find library’, try sudo port install <library name>

Install from source package:

If the above doesn’t work , try to install from source package; The port install xcrysden

step should give you most of the library dependence

  1. Download Source code :

    Source package: xcrysden-1.6.2.tar.gz

    unzip it and enter the xcrysden-1.6.2 directory

  1. copy the make file:
    #you are at source code root directory
cp ./system/Make.sys-semishared-macosx-x11-macports Make.sys

  1. try to make
    #you might need wget to make, let's install it first
sudo port install wget
#try to make
make all
  1. if there are tcl library not found error:
    1. Open the Make.sys
    #open the Make file
vi Make.sys

    b. Find the parameter TCL_LIB

    append line : -L/opt/local/lib, so this line will looks like 

    TCL_LIB      = -L/opt/local/lib -L$(TOPDIR)/external/lib -ltcl$(TCL_VER2)

    c. If this step doesn’t work, use homebrew to install the library:

    #install homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
#append environment
/opt/homebrew/bin/brew shellenv >> ~/.bash_profile
#make sure you have it 
brew --version
#use brew to install tcl
brew install tcl-tk
#now open Make file:
vi Make.sys

    append the brew library to TCL_LIB 

    TCL_LIB      = -L/opt/homebrew/lib -L$(TOPDIR)/external/lib -ltcl$(TCL_VER2)

    You might also need this lib ? 

    brew install wget
